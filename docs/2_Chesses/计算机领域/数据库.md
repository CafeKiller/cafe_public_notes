# 数据库笔记

## Redis

数据库类型：NoSQL 

一个开源的、基于内存的高性能键值存储系统，常被用作数据库、缓存和消息中间件。

主要作用：高速缓存、数据结构存储、会话管理、消息队列、分布式锁...
