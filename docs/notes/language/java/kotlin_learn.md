## Kotlin

Java 依旧是国内非常主流的编程语言，能涉及的领域也非常多，同时社区环境也是最完善的。虽然 Java 已经有不少略显老旧，令人吐槽的特点了，不过没有关系配合同 JVM 系的 Kotlin 语言，可以增强其一定的现代性能。同时 Java 也在一直更新，依旧是非常值得考虑的。

学习难度：一般，国内教程非常丰富，同时也是本人的入门语言，也算比较熟悉。
应用领域：**Web开发**、**安卓开发**、中间件开发、GUI开发
