# Golang

前几年被认为最有可能占领 Java 生态位的一门编程语言，但几年下来，确实可以补充或者说占领部分 Java 的业务场景（主要是高性能 Web 开发这方面，其他领域还没有办法对 Java 构成足够大的威胁），其有一个非常大的有点那就是语法相对于 Java 、C++ 简单很多。

学习难度：一般（属于强类型语言，语法格式相对简单）
应用领域：**高性能 Web 开发**，中间件开发
