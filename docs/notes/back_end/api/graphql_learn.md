## GraphQL

一种开源的数据查询和操作语言，同时也是用于 API 的运行时环境。

它允许客户端 精确指定需要的数据结构，解决了传统 REST API 中存在的 过度获取 和 不足获取 问题。

> [!help] 参考资料
>

>
> [中文官网 - GraphQL入门](https://graphql.cn/learn/)

主要适用场景：多端数据适配、复杂数据关系、微服务聚合层、实时应用、按需加载的低带宽环境...
