# Babel 手册

Babel 是一个 JavaScript 编译器,提供了JavaScript的编译过程，能够将源代码转换为目标代码。

> [!tip] 核心功能
> 
> - 语法转换：将新版本的 JavaScript 语法转换为旧版本的语法
> - Polyfill：通过引入额外的代码，使新功能在旧浏览器中可用
> - JSX: 将 JSX 语法转换成普通的 JavaScript 语法
> - 插件: 为 Babel 提供自定义功能
